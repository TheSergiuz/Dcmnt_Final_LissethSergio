# Especificaciones Técnicas

## Plataforma Android
- **Min SDK Version:** Android 5.0 (API 21)
- **Target SDK Version:** Android 14 (API 34)
- **Lenguaje:** Kotlin 1.9.0
- **IDE:** Android Studio Hedgehog 2023.1.1

## Arquitectura
- **Patrón:** MVVM (Model-View-ViewModel)
- **Inyección de dependencias:** Hilt 2.48
- **Base de datos local:** Room 2.6.0

## APIs y Librerías
- **Streaming de audio:** ExoPlayer 2.19.1
- **Networking:** Retrofit 2.9.0 + OkHttp 4.12.0
- **Compresión de audio:** Opus Codec (70% reducción)
- **UI Components:** Material Design 3
